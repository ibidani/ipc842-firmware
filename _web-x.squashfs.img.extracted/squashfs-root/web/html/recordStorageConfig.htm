命名帮助

普通参数

  • %02 本设备ip
  • %03 通行方向
  • %04 车道号
  • %05 速度
  • %06 限速值
  • %07 违章类型
  • %09 车牌
  • %10 车牌类型
  • %11 车辆大小
  • %12 车身颜色
  • %13 图片组总张数
  • %14 当前图片组内序号
  • %16 车牌颜色
  • %17 车牌坐标
  • %18 违章代码
  • %19 地点
  • %20 设备所属单位
  • %21 图片场景
  • %22 设备编号
  • %24 卡口编号
  • %26 车道方向
  • %57 设备编码

时间日期参数

  • %y 年
  • %h 时
  • %M 月
  • %n 分
  • %d 日
  • %s 秒
  •    
  • %S 毫秒
